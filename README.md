# PrimeraAdministracionDeBasesDeDatos

---

Este proyecto consiste en el diseño y administración de una Base de Datos relacional para la Zapateriamenjivar. Fue desarrollado como parte de mis prácticas de desarrollo de software con el objetivo de gestionar de forma eficiente el inventario, los productos y el registro de las ventas del negocio.
---

## Características del Proyecto

* Gestión de Inventario: Control de existencias de calzado por modelo, talla y precio.
* Administración de Productos: Registro de marcas y categorías, como deportivo, formal y casual.
* Control de Ventas: Registro organizado de las ventas realizadas a los clientes.
